MySimpleOS

Un sistema operativo minimalista che cresce con te

Informazioni sul Progetto

MySimpleOS è un progetto educativo che mira a creare un sistema operativo semplice ma funzionale. Partendo da un bootloader essenziale, il sistema verrà gradualmente espanso con funzionalità sempre più avanzate. L'obiettivo è comprendere i fondamenti dei sistemi operativi attraverso lo sviluppo pratico.

Roadmap del Progetto

Fase 1: Bootloader Iniziale

In Sviluppo
  • Implementazione di un bootloader minimalista
  • Avvio base con visualizzazione di testo
  • Caricamento in memoria del kernel iniziale
  • Primi test su hardware virtuale

Fase 2: Sistema Base

Pianificato
  • Kernel minimalista con gestione base della memoria
  • Interfaccia a linea di comando
  • Implementazione di comandi base (echo, help, info)
  • Driver di base per display e input

Fase 3: Prime Applicazioni

Pianificato
  • Calcolatrice semplice
  • Gioco "Snake" stile Nokia
  • App per le note di testo
  • Orologio e timer

Fase 4: Miglioramenti Grafici

Pianificato
  • Interfaccia grafica minimalista
  • Sistema di finestre semplice
  • Menu di avvio e navigazione
  • Temi e personalizzazione base

Fase 5: Applicazioni Avanzate

Pianificato
  • Browser web semplice
  • Lettore multimediale
  • Editor di testo avanzato
  • Altri giochi (Tetris, Memory)

Come Contribuire

Se sei interessato a contribuire a questo progetto, ci sono diversi modi in cui puoi aiutare:

  • Testing del bootloader e delle applicazioni
  • Sviluppo di nuove funzionalità o applicazioni
  • Miglioramento della documentazione
  • Segnalazione di bug e problemi tramite le issues di GitHub

Per iniziare, fai una fork del repository, crea un branch per la tua modifica e invia una pull request!
